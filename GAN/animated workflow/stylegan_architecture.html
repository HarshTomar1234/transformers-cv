⬡ Mapping Network f

8-layer MLP that transforms latent z into intermediate latent w. This disentangles the latent space — each dimension of w controls a more independent factor of variation than z would.

f: Z → W (8 × FC 512 → LeakyReLU)
w ∈ W is more disentangled than z ∈ Z

◆ AdaIN (Adaptive Instance Norm)

The style w is transformed via learned affine (A) into scale (y_s) and bias (y_b), which modulate feature maps at every synthesis layer via instance normalization. This is HOW style controls generation.

AdaIN(x, y) = y_s · (x − μ(x))/σ(x) + y_b
y_s, y_b = Affine(w) per layer

◈ Synthesis Network

Starts from a learned 4×4×512 constant (not noise!). Progressively upsamples through conv blocks, each receiving style via AdaIN and stochastic noise. No direct latent z input anywhere — all semantic control comes through w.

Constant 4×4 → [Conv + AdaIN + Noise] × N
Progressive: 4→8→...→1024

▲ Per-Pixel Noise

Gaussian noise is injected after each conv layer, scaled by a learned per-channel weight. Controls stochastic variation (freckles, hair strands, pores) independently from global style — separating "what" from "random details".

noise_inject = B × noise (per resolution)
B = learned scaling factor

★ Style Mixing

Use different w vectors at different layers to mix attributes. Coarse w₁ (4-8px) controls pose/shape, fine w₂ (128+px) controls color/texture. This enables interpolation, mixing faces, and disentangled attribute editing.

Coarse layers: w₁ → pose, face shape
Fine layers: w₂ → color, lighting, micro-detail

▸ Key Innovations over Progressive GAN

Mapping Network

Disentangles latent space — each w dimension controls one attribute independently

AdaIN Injection

Style injected at every layer instead of once at input — granular control over coarse vs fine features

Noise Injection

Separates stochastic variation from deterministic style — freckles don't affect face shape

Constant Input

No more feeding z directly — constant 4×4 tensor means ALL variation comes from w and noise

Style Mixing

Regularization via mixing two z→w mappings at random crossover points during training
