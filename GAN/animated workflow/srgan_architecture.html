⬡ SRResNet Generator

Deep residual network with 16 residual blocks operating at LR resolution, followed by PixelShuffle upsampling. Large 9×9 kernels at input/output capture broad context. Global skip connection aids gradient flow from input to output.

16 ResBlocks (Conv-BN-PReLU-Conv-BN + skip)
2× PixelShuffle upsamplers (total ×4)
Global skip: input features added to output

◆ PixelShuffle (Sub-Pixel Conv)

Upsamples by rearranging channels into spatial dimensions. A conv produces r² channels, then PixelShuffle reshapes H×W×(C·r²) → rH×rW×C. More efficient and artifact-free than transposed convolutions or bilinear upsampling.

Conv: H×W → H×W×(64·4) = H×W×256
Shuffle: H×W×256 → 2H×2W×64
×2 applied twice = ×4 total upscale

◈ Perceptual Loss (VGG)

The breakthrough: instead of comparing pixels directly (MSE → blurry), compare high-level feature representations from a pre-trained VGG19 network. Features from conv5_4 capture texture and structure, producing photo-realistic outputs.

L_VGG = ‖φ_5,4(I_SR) − φ_5,4(I_HR)‖₂²
φ = VGG19 feature extractor (frozen)
Captures texture, not just pixel color

▲ Residual Blocks

16 identical blocks with skip connections enable very deep networks without degradation. Each block: Conv3×3 → BN → PReLU → Conv3×3 → BN, with element-wise addition of the input. All processing happens at LR resolution for efficiency.

ResBlock(x) = x + Conv(PReLU(BN(Conv(x))))
All 16 blocks @ LR resolution (efficient)
PReLU activation (learnable slope)

★ MSE vs SRGAN Output

MSE-only training (SRResNet) achieves highest PSNR but produces over-smoothed results. SRGAN with perceptual+adversarial loss has lower PSNR but produces sharper, more photo-realistic textures that humans prefer. Quality ≠ PSNR.

SRResNet (MSE): High PSNR, blurry
SRGAN (perceptual): Lower PSNR, photo-real
Human preference: SRGAN >> SRResNet

▸ Perceptual Loss Function (3 Components)

1. Content Loss (VGG Feature Matching)

Compare VGG19 feature maps of SR and HR images at conv5_4. Captures perceptual similarity — textures, edges, structure — rather than pixel-level accuracy. The key to photo-realism.

L_content = (1/WH) ‖φ(I_SR) − φ(I_HR)‖₂²
φ = VGG19 features at conv5_4 (frozen)
2. Adversarial Loss

Standard GAN loss encouraging G to produce images that D classifies as real HR. Pushes generated images onto the manifold of natural high-resolution photographs.

L_adv = −log D(G(I_LR))
Encourages photo-realistic textures
Full Perceptual Loss

Weighted combination of content and adversarial losses. Content loss dominates (λ=1e-3 for adversarial). Optional pixel-wise MSE can be added but reduces perceptual quality.

L_perceptual = L_content + 1e-3 · L_adv
Optional: + L_MSE (but hurts perceptual quality)
