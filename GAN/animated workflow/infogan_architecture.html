⬡ Latent Codes c

Structured latent variables that the generator is forced to use meaningfully. Unlike noise z (which captures unstructured randomness), codes c automatically discover interpretable factors of variation through MI maximization — without any supervision.

c₁ ~ Cat(K=10): discrete (e.g. digit type)
c₂ ~ Unif(-1,1): continuous (e.g. rotation)
z ~ N(0,I): unstructured noise

◈ Q Network (Auxiliary)

Shares all conv layers with D (cheap — only adds one FC head). Q tries to predict back the original latent codes c from the generated image G(z,c). If Q succeeds, it proves G actually used c meaningfully in generation.

Q(c|x) approximates P(c|x)
Shared backbone with D → negligible extra cost
Cat → Softmax, Cont → Gaussian (μ, σ)

◆ Mutual Information I(c; G(z,c))

Maximizing I(c; G(z,c)) ensures c is not ignored by G. Since MI is hard to compute directly, InfoGAN uses a variational lower bound via Q network. The resulting loss is just cross-entropy between input c and Q's prediction.

I(c; G(z,c)) ≥ E[log Q(c|G(z,c))] + H(c)
H(c) is constant → maximize E[log Q(c|G(z,c))]
= Cross-entropy between c and Q's output

★ Discovered Representations

On MNIST: c₁ (categorical) learns digit identity (0-9), c₂ learns rotation angle, c₃ learns stroke width — all completely unsupervised. On CelebA: discovers glasses, hair color, pose, emotion. Disentangled and interpretable.

MNIST examples (unsupervised!):
c₁ Cat(10) → digit identity (0-9)
c₂ Cont → rotation angle
c₃ Cont → stroke width

▸ Types of Latent Codes

Categorical c₁

Discrete codes sampled from Categorical distribution. Discover mutually exclusive categories (digit type, object class). Q outputs softmax probabilities over K classes.

c₁ ~ Cat(K=10)
Loss: Cross-entropy
Discovers: discrete factors
Continuous c₂

Continuous codes sampled from Uniform(-1,1). Discover smooth, continuous variations (angle, thickness, lighting). Q models as Gaussian and outputs μ, σ parameters.

c₂ ~ Uniform(-1, 1)
Loss: Gaussian log-likelihood
Discovers: continuous factors
Noise z (incompressible)

Standard GAN noise that captures remaining unstructured variation. Not regularized by MI — free to encode anything. Acts as randomness source for variation not captured by c.

z ~ N(0, I), z ∈ ℝ⁶²
No MI constraint on z
Captures: unstructured randomness
