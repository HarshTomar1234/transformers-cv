⬡ U-Net Generator

Encoder-decoder with skip connections that concatenate encoder feature maps to decoder at each resolution level. This lets the network pass low-level details (edges, colors) directly across, while the bottleneck handles high-level semantics.

Encoder: Conv↓ × 8 (256→1)
Decoder: DeConv↑ × 8 + skip concat (1→256)
Dropout(0.5) in first 3 decoder layers

◈ PatchGAN Discriminator

Instead of classifying the whole image as real/fake, PatchGAN classifies each 70×70 overlapping patch independently. This acts as a learned texture/style loss, penalizing unrealistic local structure while L1 handles global coherence.

Input: concat(x, y) → 256×256×6
Output: 30×30 matrix of patch scores
Conditional: sees source AND target

◆ Skip Connections

The defining feature of U-Net. Each encoder layer's output is concatenated (channel-wise) with the corresponding decoder layer, doubling channel count. This shuttles fine spatial info across the bottleneck — essential for pixel-aligned tasks.

dec_i = concat(enc_{n-i}, up(dec_{i-1}))
Channels double at each skip level

★ Paired Training Data

Requires aligned input-output pairs: {source, target}. Examples: edges→shoes, segmentation→photo, satellite→map, BW→color. The pairing provides direct supervision that enables pixel-accurate translation.

Training pairs: {x_i, y_i}
e.g. {edge_map, photo} aligned pixel-to-pixel

▸ Objective: cGAN Loss + L1 Reconstruction

Conditional Adversarial Loss

Standard cGAN loss — D sees both the source input x and the output (real target or generated). G tries to fool D into thinking G(x) is the real target for input x. Ensures outputs look realistic.

L_cGAN = E[log D(x, y)] + E[log(1 − D(x, G(x)))]
L1 Reconstruction Loss

Direct pixel-wise L1 distance between generated output G(x) and real target y. Encourages outputs to be close to ground truth, handling global structure while D handles local texture. L1 chosen over L2 for less blurring.

L_L1 = E[‖y − G(x)‖₁]
Full: G* = arg min_G max_D L_cGAN + λ·L_L1
λ = 100 (L1 heavily weighted)
