Forward Cycle: A → G(A) → F(G(A)) ≈ A

Backward Cycle: B → F(B) → G(F(B)) ≈ B

⬡ Two Generators (G, F)

G maps A→B, F maps B→A. Both use ResNet-based encoder-decoder architecture with 9 residual blocks, InstanceNorm, and ReLU. The encoder downsamples, residual blocks transform, and the decoder upsamples with transposed convolutions.

G: A → B (e.g. horse → zebra)
F: B → A (e.g. zebra → horse)

◈ Two Discriminators (D_A, D_B)

D_A distinguishes real A from F(B). D_B distinguishes real B from G(A). Both use PatchGAN architecture — classifying 70×70 overlapping patches rather than the whole image, which captures local texture effectively.

PatchGAN: 70×70 patches
Output: N×N matrix of real/fake scores

◆ Cycle Consistency Loss

The key innovation: if you translate A→B→A, you should get back the original A. This constraint prevents mode collapse and preserves content without requiring paired training data. Uses L1 distance.

L_cyc = ‖F(G(x_A)) − x_A‖₁ + ‖G(F(x_B)) − x_B‖₁
λ_cyc = 10

★ No Paired Data Required

Unlike Pix2Pix, CycleGAN needs NO aligned image pairs — just two unrelated collections from domain A and B. Cycle consistency acts as a self-supervision signal, learning the mapping purely from distribution-level information.

Training: {x_A} from A, {x_B} from B
No pairing: horse₁ ≠ zebra₁

▸ Full Objective (4 Components)

1. Adversarial Loss (G, D_B)

Standard GAN loss for G: A→B translation. D_B tries to distinguish G(x_A) from real x_B. Uses LSGAN (MSE) loss for stability.

L_GAN(G, D_B) = E[(D_B(G(x_A)) − 1)²]
2. Adversarial Loss (F, D_A)

Standard GAN loss for F: B→A translation. D_A tries to distinguish F(x_B) from real x_A.

L_GAN(F, D_A) = E[(D_A(F(x_B)) − 1)²]
3. Cycle Consistency Loss

Round-trip reconstruction in both directions. Ensures content preservation through the translation cycle.

L_cyc = ‖F(G(x_A)) − x_A‖₁ + ‖G(F(x_B)) − x_B‖₁
4. Identity Loss (Optional)

If you feed a real B image to G (A→B), it should remain unchanged. Helps preserve color composition.

L_idt = ‖G(x_B) − x_B‖₁ + ‖F(x_A) − x_A‖₁
λ_idt = 0.5 × λ_cyc
