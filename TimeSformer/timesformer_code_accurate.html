Pre-LN Pattern
LayerNorm is applied before each sub-block. The normalized output goes through the sub-block, then is added to the original input via residual connection. This is the standard approach used in the original TimeSformer paper.
x = x + sub_block(norm(x))
Temporal Block
Rearrange → (b n) t d, run self-attn across T frames, rearrange back.
xt = self.norm1(x)
xt = rearrange(xt, "b t n d → (b n) t d")
xt = temporal_attn(xt, xt, xt)[0]
x = x + xt
Spatial Block
Rearrange → (b t) n d, run self-attn across N patches within each frame, rearrange back.
xs = self.norm2(x)
xs = rearrange(xs, "b t n d → (b t) n d")
xs = spatial_attn(xs, xs, xs)[0]
x = x + xs
MLP Block
Linear → GELU → Linear. Expands to 4× dim then projects back. Same Pre-LN pattern.
x = x + mlp(norm3(x))

Pre-LN is preferred for deeper models — it provides better gradient flow and more stable training.
